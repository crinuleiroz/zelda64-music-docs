# Bank 0x28

:::note

    This instrument bank is a copy of Bank 0x26. However, it uses sample bank 0 instead of sample bank 2, so its sample pointers load incorrect VADPCM data from memory.

:::

## Sequences

:::info

    No sequences use this instrument bank.

:::
