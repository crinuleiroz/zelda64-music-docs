# Bank 0x25

:::note

    This instrument bank is a copy of Bank 0x04. However, it uses sample bank 0 instead of sample bank 2, so its sample pointers load incorrect VADPCM data from memory.

:::

## Sequences

:::info

    No sequences use this instrument bank.

:::
